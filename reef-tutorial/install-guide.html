REEF & YARN Installation Guide

Install Java

I recommend to install HotSpot version of Java 7 (or above).
  • (OSX) 
            sudo ln -s /usr/bin/java /bin/java

Install Hadoop

  • Enable SSH 
            ssh localhost # (1) If it works, we can skip SSH part
        ls ~/.ssh/id_rsa # (2) If a key file exists, go to (4)
        ssh-keygen -t rsa -P '' -f ~/.ssh/id_rsa # (3) Do not copy & paste. Write on your own :)
        cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys # (4)
        ssh localhost # it works now?
  • Download Hadoop & Set path variables in the profile (e.g. ~/.profile) 
            export HADOOP_HOME = {dir_you_installed}
        export YARN_HOME = {dir_you_installed}
        source ~/.profile # Or you may use ~/.bash_profile or ~/.bashrc
  • Format namenode 
            $HADOOP_HOME/bin/hadoop namenode -format
  • Check hadoop successfully installed 
            $HADOOP_HOME/sbin/start-dfs.sh
        $HADOOP_HOME/sbin/start-yarn.sh
  • You can see it working via web interface 
            {Namenode} http://localhost:50070
        {YARN} http://localhost:8088
  • Run an example application 
            yarn jar $HADOOP_HOME/share/hadoop/yarn/hadoop-yarn-applications-distributedshell-*.jar $HADOOP_HOME/share/hadoop/yarn/hadoop-yarn-applications-distributedshell-*.jar --shell_command date --num_containers 2

Running Examples

Now we can run REEF example on YARN environment. 
    cd {REEF_DIR}/reef-examples # REEF_DIR: Where you installed REEF
    yarn jar target/reef-examples-*-shaded.jar org.apache.reef.examples.hello.HelloReefYarn
