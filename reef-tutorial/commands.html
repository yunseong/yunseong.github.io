URL links and Commands

This page is intended to support all the URL links and commands used in the REEF tutorial, especially for the Setting up the environment section.
If you use Ubuntu, you may want to use a shell script made by Taegeon. You can take a look at it here

Install Java 7

  • (OSX)
    Download jdk 7 installer at here
    export JAVA_HOME=$(/usr/libexec/java_home -v 1.7)
  • (Linux)
    sudo apt-get install openjdk-7-jdk
    export JAVA_HOME=/usr/lib/jvm/java-1.7.0-openjdk-amd64 && echo export JAVA_HOME=$JAVA_HOME >> ~/.profile

Install Hadoop

  • Enable SSH

    ssh-keygen –t rsa –P ‘’ –f ~/.ssh/id_rsa
    cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
    ssh localhost # see if it works
    exit

  • Download Hadoop

    wget http://mirror.apache-kr.org/hadoop/core/hadoop-2.2.0/hadoop-2.2.0.tar.gz
    cd {dir_to_install} (your home is OK)
    tar xzvf hadoop-2.2.0.tar.gz
    Set path variables in your profile (~/.profile)
    export HADOOP_HOME = {dir_you_installed}
    export YARN_HOME = {dir_you_installed}

  • Check hadoop successfully installed

    HADOOP_HOME/sbin/start-dfs.sh
    HADOOP_HOME/sbin/start-yarn.sh

  • You can see it working via web interface

    Namenode – http://localhost:50070
    YARN – http://localhost:8088

  • Run an example application

    hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-2.2.0.jar pi 20 5

Install Git

  • (Linux) sudo apt-get install git
  • (OSX) brew install git
  • git --version

Install Maven

  • (Linux) sudo apt-get install maven
  • (OSX) brew install maven
  • mvn -version # Should be v3.x

Install Protocol Buffer

  • (Linux) https://code.google.com/p/protobuf/
  • (OSX) brew install protobuf
  • protoc --version # Should be v2.5.0

Build REEF

  • git clone https://github.com/Microsoft-CISL/TANG.git
  • cd TANG
  • mvn -T1C -q -fae -DskipTests clean install eclipse:clean eclipse:eclipse package

Running Examples

  • cd $REEF_HOME/reef-examples
  • HelloREEF : mvn -PHelloREEF
  • RetainedEval : mvn -PRetainedEval
  • StatePassing : Part of REEF-tests
  • GroupCommunication : Not working currently
