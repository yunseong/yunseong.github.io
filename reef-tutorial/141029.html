REEF Tutorial

Commands

Install Java

  • (OSX)

    sudo ln -s /usr/bin/java /bin/java

Install Hadoop

  • Enable SSH

    ssh localhost # see if it works. If it does not work, ssh-keygen –t rsa –P ‘’ –f ~/.ssh/id_rsa
    cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
    exit

  • Download Hadoop & Set path variables in the profile (e.g. ~/.profile)

    export HADOOP_HOME = {dir_you_installed}
    export YARN_HOME = {dir_you_installed}
    source ~/.profile

  • Format namenode

    $HADOOP_HOME/bin/hadoop namenode -format

  • Check hadoop successfully installed

    $HADOOP_HOME/sbin/start-dfs.sh
    $HADOOP_HOME/sbin/start-yarn.sh

  • You can see it working via web interface

    {Namenode} http://localhost:50070
    {YARN} http://localhost:8088

  • Run an example application

    yarn jar $HADOOP_HOME/share/hadoop/yarn/hadoop-yarn-applications-distributedshell-2.4.0.jar org.apache.hadoop.yarn.applications.distributedshell.Client --jar $HADOOP_HOME/share/hadoop/yarn/hadoop-yarn-applications-distributedshell-2.4.0.jar --shell_command date --num_containers 2

Running Examples

  • Go to {REEF_DIR}/reef-examples

  • HelloREEFYarn

    yarn jar target/reef-examples-0.10-incubating-SNAPSHOT-shaded.jar org.apache.reef.examples.hello.HelloReefYarn

  • RetainedEval

    yarn jar target/reef-examples-0.10-incubating-SNAPSHOT-shaded.jar org.apache.reef.examples.retained_eval.Launch
